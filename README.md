# UI Component Playground

Учебный pet-проект на React/Next.js, в котором я отрабатываю построение интерфейсов, работу с глобальным state(Redux) и организацию фронтенд-архитектуры на реальном UI-сценарии со списком карточек.

## О проекте

В проекте реализована страница карточек с управлением состоянием через Redux Toolkit.  
Основная цель проекта — не просто собрать интерфейс, а потренировать подход к структуре приложения: slices, selectors, memoized selectors, фильтрация, пагинация, кастомные компоненты и стилизацию через SCSS Modules.

## Реализовано

- страница карточек
- фильтрация по категории
- поиск по названию
- пагинация
- кастомный Select
- глобальный state на Redux Toolkit
- memoized selectors через `createSelector`
- стилизация через SCSS Modules
- адаптация структуры проекта под дальнейшее расширение

## Стек

- Next.js
- React
- TypeScript
- Redux Toolkit
- SCSS Modules

## Архитектурные идеи

В проекте я отдельно отрабатываю:
- разделение UI и логики
- хранение глобального состояния в Redux Toolkit
- работу с selectors и derived state
- memoization через `createSelector`
- отдельный state представления списка карточек (`cardsViewSlice`)
- постепенный рефакторинг компонентов по мере роста логики
